###Instruction to extract homing barcodes from fastq files.

Step 1: Unzip the NSCseq_data_processing file in the folder where the fastq files are located.

Step 2: Open linax command line window and make that fastq containing folder as directory

Step 3: run bash script as: 
$ bash loop.sh 
CSV file will be saved as output folder.

Step 4: Use the CSV file as input for NSCseq_data_processing.R

Step 5: Save the mutation matrix.
